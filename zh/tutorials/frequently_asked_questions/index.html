    1. 尝试将配置文件中train.dataset.use_minddata和eval.dataset.use_minddata的选项设置为True。MindOCR将采用MindSporeMindData执行部分数据预处理步骤：

      ...

    label_file: ic15/det/train/det_gt.txt
    sample_ratio: 1.0
    use_minddata: True                          <-- 设置该选项
...
eval:
  ckpt_load_path: tmp_det/best.ckpt
  dataset_sink_mode: False
  dataset:
    type: DetDataset
    dataset_root: /data/ocr_datasets
    data_dir: ic15/det/test/ch4_test_images

    sample_ratio: 1.0

...

      ...
train:
  ...

...
